# Schichtkalender
Schichtkalender für´s Smartphone

WebCal:// Adresse im Browser des Smartphone öffnen bestätigen und schon habt ihr den fertigen Schichtkalender in euer Smartphone importiert!
iOS kann QR-Code Scannen dieser enthält die Schicht Logik für das generierte Jahr und importiert sie zu eurem iOS Standard Kalender
für Android ist ein Workaround folgender
aCalendar aus dem GooglePlay Store runter laden
QR-Code scannen
Datei anschließend öffnen beim öffnen der *.ics werdet ihr nach der APP gefragt mit der sie geöffnet werden soll
dort wählt ihr aCalendar aus und unten rechts in der Ecke dann mit dem Haken den Import bestätigen. 
aCalendar synchronisiert anschließend automatisch zu eurem Standard Kalender auf Android
